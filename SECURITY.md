
# Security Policy

The YLCA team takes security and responsible disclosure seriously.

---

## Reporting a Security Issue

If you believe you have discovered a security vulnerability related to YLCA Agent:

- ❌ Do NOT open a public GitHub issue
- ❌ Do NOT share exploit details publicly

Instead, contact the project maintainers through the private security contact
provided in the project communication channels.

---

## Scope

This policy applies to:

- YLCA Agent runtime behavior
- Deployment and execution modes
- Data handling and exposure risks

This policy does NOT apply to:

- Third-party models or libraries
- User-provided code executed by the agent
- Infrastructure managed by the user

---

## Responsible Disclosure

Please allow reasonable time for investigation and remediation before any public disclosure.

We appreciate responsible security research and coordinated disclosure.
